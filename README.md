# Decode Morse Code Message
> This is a Ruby project that decodes a Morse code character, takes a string parameter, and return the corresponding character in uppercase (e.g. `decode_char(".-")` returns `"A"`).

## Built with

- Ruby

## Getting Started

Clone the repository using `git clone` then open terminal and enter `cd morse_code`
